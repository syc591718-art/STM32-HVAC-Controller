# 多分区智能 HVAC 温控系统

基于 STM32F103 的智能温控系统，通过 UART 自定义协议与 PyQt5 上位机通信，实现 7 个分区的环境监控与闭环控制。

## 技术栈

| 类别 | 技术 |
| :--- | :--- |
| 主控 | STM32F103 |
| 传感器 | DHT11（温湿度） |
| 显示 | LCD 显示屏（480×320） |
| 通信 | UART 自定义协议 |
| 上位机 | PyQt5 + PySerial |
| 编程语言 | C / Python |
| 开发工具 | Keil MDK |

## 功能

### 嵌入式端
- 7 个分区温湿度采集与控制（支持真实传感器 + 模拟演示）
- UART 自定义协议（帧头 `<<` + ID + CMD + D1 + D2 + 校验和 + 帧尾 `>>`）
- 自动调节逻辑：
  - 温度过高 → 制冷，温度过低 → 制热
  - 湿度过高 → 开启风机
- LCD 界面显示（房间卡片、温湿度数值、空调/风机状态）
- 按键交互：
  - 切换选中房间
  - 手动调节温度上下限
- LED 状态指示：
  - 红灯：制热，蓝灯：制冷，绿灯：恒温
  - 风机状态 LED

### 上位机端
- PyQt5 跨平台监控界面
- 实时接收 7 个分区的温湿度数据
- 下发控制命令（调节温度上下限等）

## 工程结构

| 文件夹 | 说明 |
| :--- | :--- |
| CORE/ | 内核相关文件（启动文件） |
| FWLIB/ | STM32 标准库 |
| HARDWARE/ | 硬件驱动（DHT11、LCD、按键、LED） |
| SYSTEM/ | 系统工具（延时、串口） |
| USER/ | 主程序、中断配置 |

## 通信协议

| 字节 | 内容 | 说明 |
| :--- | :--- | :--- |
| 0-1 | `0x3C 0x3C` | 帧头 `<<` |
| 2 | ID | 房间号（1-7） |
| 3 | CMD | 空调+风机状态（高4位空调，低4位风机） |
| 4 | D1 | 温度值 |
| 5 | D2 | 湿度值 |
| 6 | SUM | 校验和（前4字节累加取低8位） |
| 7-8 | `0x3E 0x3E` | 帧尾 `>>` |
